Explain plan for a SQL Query¶

Basic Terminology¶

Timeron

a unit of measure for time that CANNOT be converted to a real unit of time. It is used for comparing the relative performance of the same query (or queries) as changes are made when we look at explain plans.

Exaplain Plan

Explain plan let us see how DB2 plans to get the results that our DML SQL asked for. It shows in every bit of data, every object, every join, every sort in excruciting detil. It contains everything that is needed to retrieve and return the data.
