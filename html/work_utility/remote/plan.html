plan:: A tool allowed to quickly access explain plan of SQL¶

Last Update: February 03, 2016

Download¶

go to here

Usage¶

  1. put plan.sh into your $HOME/bin directory.

  2. connect to a local db, say feddb

  3. db2 -tvf $HOME/sqllib/misc/EXPLAIN.DDL

  4. make a sql file (i.e. test.clp) and put sql you want to explain in it.

    For example:

    In test.clp, I write the following:

    code-block:: sql

    select count(*) from (select c1, c2, c3 from n1 group by c1, c2, c3);

  5. $HOME/bin/plan.sh test.sql -db feddb
